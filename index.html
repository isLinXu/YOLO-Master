Dynamic Intelligence for Vision

Existing Real-Time Object Detection (RTOD) methods commonly adopt YOLO-like architectures for their favorable trade-off between accuracy and speed. However, these models rely on static dense computation that applies uniform processing to all inputs.

YOLO-Master introduces instance-conditional adaptive computation via Efficient Sparse Mixture-of-Experts (ES-MoE) blocks. It dynamically allocates computational resources according to scene complexity, outperforming state-of-the-art models on large-scale benchmarks.
